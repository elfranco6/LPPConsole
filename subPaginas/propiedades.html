Propiedades CSS

En esta sección vamos a ver algunas propiedades de css tales como:

propiedad valores Explicación
class N/A La propiedad de class son los más utilizados, este selector es que en una misma página HTML varios elementos diferentes pueden utilizar el mismo valor en el atributo class

 Posiciona los elementos de forma flotante

left: la caja se desplaza todo lo posible a la izquierda de la posición en la que se encontraba

right: la caja se desplaza todo lo posible a la derecha de la posición en la que se encontraba.

none: la caja no se desplaza respecto de su posición original.

inherit: se puede utilizar el valor inherit para forzar a que se produzca la herencia automática

Controla el comportamiento de los contenidos que no caben en su elemento contenedor

visible: que es el comportamiento que hace que los contenidos se salgan del elemento y sean completamente visibles

hidden: que hace que el navegador oculte todos los contenidos que se salen de la caja del elemento

scroll: que hace que el navegador muestre barras de scroll sobre un elemento cuando sus contenidos no caben en la caja del elemento

inherit: se puede utilizar el valor inherit para forzar a que se produzca la herencia automática.

Establece algunas o todas las propiedades del color y/o imagen de fondo de un elemento

color: Establece el color de fondo

repeat: Establece si la imagen se repite o no

image: Establece una imagen de fondo

position: Establece la posición de la imagen

attachment: Establece si la imagen va ser fija o no
