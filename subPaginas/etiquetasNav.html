Etiquetas de navegación

Las etiquetas de navegación son etiquetas que son ultizadas para la navegacion del un sitio web

estas etiqueas pueden ser

Etiqueta Atributo Valor Explicación

nav

 Class, id Este elemento no tiene atributos que no sean los atributos globales, comunes a todos los elementos.

Esta etiqueta define un conjunto de enlaces de navegación

la etiqueta nav solo se utiliza para un bloque principal de enlaces de navegación.
ul Class, id Este elemento no tiene atributos que no sean los atributos globales, comunes a todos los elementos. ul significa listas desordenadas estas cuando se usan generalmente tienen unas viñetas del lado derecho
ol Class, id no tiene valor especifico, el desarrollador es el encagado de poner valor al atributo. ol significa listas ordenadas estas cuando se usan generalmente tienen números en al lado izquierdo
li class,id no tiene valor especifico, el desarrollador es el encagado de poner valor al atributo. li significa ítem de lista, esta lista puede ser usadas en listas ordenadas ol o listas desordenadas ul
